## 📦 Cài đặt & Chạy thử

```bash
# Clone repo
git clone https://github.com/hiimcoder21-056/gift-card
# Di chuyển vào thư mục
cd gift-card

# Cài đặt dependencies
npm install hoặc yarn add

# Chạy dev
npm run dev hoặc yarn dev

 ```
